
*******************
SYSTEM REQUIREMENTS
*******************

Most tools require:

\* Perl v5.8 or newer

\* Bash v3 or newer

\* Core Perl modules like Time::HiRes

Tools that connect to MariaDB require:

\* Perl modules DBI and DBD::mysql

\* MariaDB 10.3+

MariaDB Tools officially supports and is tested on many popular Linux
distributions and MariaDB 10.3+.
